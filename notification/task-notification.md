# 任务通知

云信推送的任务消息

## 发布任务通知

```json
{
    "type": 2000,
    "body": {
        "id": 20,
        "name": "任务发布成功",
        "title": "电脑开不开机，黑屏",
        "user": "碧云天",
        "remark": "您的任务已成功发布",
        "created_at": "2019-11-25 14:37:11"
    }
}
```
