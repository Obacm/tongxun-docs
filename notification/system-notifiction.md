# 任务通知

云信推送的系统消息

## 充值到账通知

```json
{
    "type": 1010,
    "body": {
        "id": 20,
        "name": "充值成功",
        "amount": "30.00",
        "remark": "您的充值已到账",
        "created_at": "2019-11-25 14:37:11"
    }
}
```

## 任务退款到账通知

```json
{
    "type": 1052,
    "body": {
        "id": 20,
        "name": "退款到账提醒",
        "actual_amount": "30.00",
        "remark": "您的任务打赏已退回原账户",
        "created_at": "2019-11-25 14:37:11"
    }
}
```
