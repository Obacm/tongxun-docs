## 任务邀约

```json
{
    "type": 202,
    "data": {
        "id": 10,
        "task_id": 20,
        "title": "任务申请",
        "content": "碧云天已向您发起了任务邀约申请，请您确认是否接受该申请。"
    }
}
```

## 任务邀约同意

```json
{
    "type": 203,
    "data": {
        "title": "已接受您的邀约"
    }
}
```

## 任务邀约拒绝

```json
{
    "type": 204,
    "data": {
        "title": "已拒绝您的邀约"
    }
}
```
