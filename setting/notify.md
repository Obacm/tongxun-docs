# 用户通知设置

通知消息设置

## 请求地址

> `setting/notify`

## 请求类型

> `POST`

## 请求参数

#### Header

> - `Authorization` `Bearer access_token` 授权token

#### Body

> - `system_notify` `int` 系统消息  `[0, 1]`
> - `task_notify` `int` 任务消息  `[0, 1]`
> - `qa_notify` `int` 问答消息  `[0, 1]`
> - `im_notify` `int` 聊天消息  `[0, 1]`

## 响应示例

```json
{
    "ok": 1
}
```

## 异常示例

> - `system_notify 必须为布尔值`

```json
{
    "errno": 20001,
    "msg": "system_notify 必须为布尔值",
    "ok": 0
}
```
