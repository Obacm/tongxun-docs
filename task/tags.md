# 评论标签

获取评分对应的标签

## 请求地址

> `task/comment/tags`

## 请求类型

> `GET`

## 请求参数

> - `score` `int` 分数 `1, 2, 3, 4, 5`
> - `type` `int` 评论类型 `1: 发布者->接取者的评论, 2: 接取者->发布者的评论`

## 响应参数

> - `name` `string` 标签

## 响应示例

```json
{
    "data": [
        {
            "name": "服务态度好"
        },
        {
            "name": "工作效率"
        },
        {
            "name": "有耐心"
        },
        {
            "name": "随叫随到"
        }
    ],
    "ok": 1
}
```
