# 客服

聊天框里任务分享

## 请求地址

> `task/[id]/customer`

## 请求类型

> `POST`

## 请求参数

#### Header

> - `Authorization` `Bearer access_token` 授权token

## 响应示例

```json
{
    "ok": 1
}
```
